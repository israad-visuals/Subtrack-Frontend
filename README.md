# SubTrack - Frontend

## About
Frontend web application for SubTrack, a personal finance and subscription management dashboard. Built for COMP 2154 System Development Project at George Brown College.

## Project Description
SubTrack helps users track recurring subscriptions and free trials, avoid forgotten charges, and understand their monthly spending through a visual "Traffic Light" dashboard (Red = due soon, Yellow = upcoming, Green = safe, Blue = free trial).

## Tech Stack
- HTML5
- CSS3
- JavaScript
- React.js 

## Features
- User Login and Registration
- Dashboard with Traffic Light subscription urgency system
- Monthly Burn Rate calculator display
- Add / Edit / Cancel subscriptions
- Free Trial tracking with expiry warnings
- Category-based subscription filtering
- Responsive design (desktop + mobile)
